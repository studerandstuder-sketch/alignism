This is a minimal Next.js 14 + TailwindCSS project for alignism.org.

TypeScript version has been set to 5.3.3 to avoid Netlify ETARGET errors.

## Usage

1. Upload this folder to a new GitHub repository.
2. On your local machine:
   - Clone the repo
   - Run: `npm install`
   - Run: `npm run dev` to test locally
3. Deploy to Netlify:
   - Build command: `npm run build`
   - Publish directory: `.next`
